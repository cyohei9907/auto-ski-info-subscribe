## 📊 功能展示

### 1. 统计卡片（未选中状态）

```
┌─────────────┐
│ 30分間隔     │  ← 灰色边框
│             │
│      2      │  ← 黑色数字
└─────────────┘
     ↑
  可点击
```

### 2. 统计卡片（选中状态）

```
┏━━━━━━━━━━━┓
┃ 30分間隔     ┃  ← 蓝色边框
┃             ┃
┃      2      ┃  ← 蓝色数字
┗━━━━━━━━━━━┛

## 🎬 操作流程演示

### 流程 1：查看特定间隔的账户

```
步骤 1: 初始状态
┌────────┐ ┌────────┐ ┌────────┐
│ 30分: 2│ │ 1時: 1 │ │ 4時: 2 │  ← 所有卡片默认样式
└────────┘ └────────┘ └────────┘

        ↓ 点击 "30分間隔" 卡片

步骤 2: 筛选激活
┏━━━━━━━━┓ ┌────────┐ ┌────────┐
┃ 30分: 2┃ │ 1時: 1 │ │ 4時: 2 │  ← 选中卡片高亮

## 🚀 性能指标

- **统计计算**: < 1ms（本地计算）
- **筛选响应**: < 10ms（前端筛选）
- **批量更新**: 根据选中数量（并行请求）
- **单个更新**: < 200ms（API 响应）

## 📝 使用建议

### ✅ 推荐做法

1. 使用统计卡片快速了解账户分布
2. 点击卡片筛选特定间隔的账户
3. 批量操作节省时间
4. 定期检查和调整监控间隔

### ❌ 避免的做法

1. 不要频繁批量修改（会产生多个 API 请求）
2. 不要同时打开多个筛选（只支持单一筛选）
3. 不要在批量操作时刷新页面
